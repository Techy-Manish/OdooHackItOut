🏢 HRMS - Human Resource Management System

An HRMS (Human Resource Management System) developed for the Odoo Hackathon to simplify employee management, attendance tracking, leave requests, payroll management, and employee profile handling through an intuitive web interface.

---

 📌 Features

 👤 Authentication
- Employee & Admin Login
- User Registration
- Role-based Access
- Session Management using Local Storage

 📊 Employee Dashboard
- Dashboard Overview
- Attendance Statistics
- Quick Action Buttons
- Recent Activities
- Company Announcements

📅 Attendance Management
- Check In / Check Out
- Attendance Records
- Attendance Status

 📝 Leave Management
- Apply for Leave
- Leave History
- Leave Status Tracking

 💰 Payroll
- Salary Details
- Payroll Summary
- Monthly Salary Overview

👤 Employee Profile
- View Employee Details
- Update Profile Information

 👨‍💼 Admin Panel
- Employee Management
- Leave Approval
- Attendance Monitoring
- Payroll Overview

 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Techy-Manish/OdooHackItOut.git
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm start
```

Open

```
http://localhost:3000
```
