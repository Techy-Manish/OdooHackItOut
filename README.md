🚀 Odoo HRMS

Every Workday, Perfectly Aligned

Odoo HRMS is a modern Human Resource Management System developed for the Odoo Hackathon. It helps employees and HR administrators manage daily HR operations through a clean, responsive, and user-friendly interface.

✨ Features

- 🔐 Secure Login & Registration
- 👤 Employee Profile Management
- ⏰ Attendance (Check-In / Check-Out)
- 📅 Leave Application & Tracking
- 💰 Payroll Overview
- 🧑‍💼 HR Admin Portal
- 🌙 Light & Dark Theme
- 📱 Responsive Design

🛠️ Tech Stack

- 🌐 HTML5
- 🎨 CSS3
- ⚡ JavaScript (ES6)
- 💾 Local Storage
